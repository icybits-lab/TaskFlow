## How to Use

1. **Add a Task**:
   - Fill in the task title (required)
   - Add optional description
   - Select category and priority
   - Set a due date (optional)
   - Click "Add Task"

2. **Manage Tasks**:
   - **Complete**: Click anywhere on a task or the checkbox
   - **Edit**: Click the edit button (pencil icon)
   - **Delete**: Click the delete button (trash icon)
   - **Select**: Check the checkbox for bulk operations

3. **Filter Tasks**:
   - Use category buttons on the left
   - Use filter buttons (All, Pending, Completed, Due Today)
   - Use priority filters
   - Search using the search box

4. **View Statistics**:
   - Click the chart icon in the header
   - View completion rates and analytics
   - See category and priority distributions
